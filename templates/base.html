About

Scratching an Itch

This is a simple webpage that helps you count words, 2-word phrases (bigrams) and 3-word phrases (trigrams) in plain text. We created it to support some activities we do in our data visualization and storytelling classrooms. In our classes we introduce non-programmers to data analysis and storytelling. We want to help unlock the power of digital tools for everyone.

Empowering Everyone

There are a number of computer-assisted data analysis techniques like this that are readily available to programmers. Some of these are even just one line of code, but that one line is incomprehensible if you don't know how to program. We built this site to help bridge that gap, so non-programmers can take advantage of these capabilities.

Privacy

We take your privacy seriously. This site uses https for more secure transfer of your data. We don't log or archive the text you upload. We delete your uploaded file as soon as we have analyzed it. We won't ever use the aggregate results of analysis either. We do log that you visited, to see how popular we are.

Open Source

We built this using a number of free tools, so this is also open-source. You can download and change this however you want at our Github repository.
